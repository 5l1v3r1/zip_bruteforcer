# zip_bruteforcer
GUI program which bruteforce zip files on Python3                      
# INSTALL:                                      
git clone https://github.com/zertmark/zip_bruteforcer.git                           
cd zip_bruteforcer && chmod +x zip_bruteforcer && pip3 install zipfile                                       
# RUN:                                
./zip_bruteforcer
